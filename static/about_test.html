About this site

This is a fan-made site archiving the robots drawn by @smolrobots on Twitter! 🤖

It is a continuation of the wonderful @aguitarpenter's original archive site, so special thanks to him for starting this whole thing!!

Reporting issues

If you find something wrong with the site, please open an issue on Github.

Setting up your own archive instance

This is an open-source project; the source code can be found on Github. You are free to clone the repository and set up your own instance of this archive!
